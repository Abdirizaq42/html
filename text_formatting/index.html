Html description list

HTML
HyperText Markup Language used to structure web content.
CSS
Cascading Style Sheets used to style and layout web pages.
JavaScript
A programming language that adds interactivity to websites.
DOM
Document Object Model, a tree-like representation of an HTML document.
Semantic HTML
HTML elements that clearly describe their meaning and purpose.
Accessibility (A11y)
Practices that make websites usable by people with disabilities.
SEO
Search Engine Optimization, improving a site's visibility in search results.
Responsive Design
A design approach that adapts layouts to different screen sizes.
Viewport
The visible area of a web page within a browser window.
API
Application Programming Interface used for communication between software systems.
HTTP
HyperText Transfer Protocol used for data communication on the web.
HTTPS
A secure version of HTTP that encrypts transmitted data.
URL
Uniform Resource Locator, the address of a resource on the internet.
Frontend
The client-side part of a web application that users interact with.
Backend
The server-side logic, databases, and services powering an application.
Framework
A collection of tools and conventions that simplify software development.
Git
A distributed version control system used to track code changes.
Repository
A storage location containing a project's files and version history.
Cookie
Small pieces of data stored in a user's browser by a website.
Web Hosting
A service that makes websites accessible on the internet.
